# Apaixonados por Focinhos - Identificação Pet 🐾

Este projeto é uma landing page responsiva para a **Apaixonados por Focinhos**, uma loja especializada em produtos de identificação, bandanas e mimos para pets e seus tutores.

## 🚀 Sobre o Projeto

A página foi desenvolvida para apresentar a importância da identificação pet e facilitar a navegação dos clientes através de seções informativas e de conversão.

### Principais Seções:
- **Hero/Destaque**: Introdução visual impactante com call-to-action.
- **Por que Identificar**: Seção educativa sobre a segurança dos pets.
- **Diferenciais**: Detalhes sobre a qualidade e proposta da marca.
- **Depoimentos**: Prova social de clientes satisfeitos.
- **FAQ**: Respostas para as dúvidas mais frequentes.
- **Painel Administrativo**: Área de login restrita para gestão (modal integrado).

## 🛠️ Tecnologias Utilizadas

O projeto utiliza tecnologias modernas de front-end para garantir performance e responsividade:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3 (Custom Properties)**: Estilização personalizada utilizando variáveis para paleta de cores (Rosa, Azul, Azul Claro).
- **Bootstrap 5**: Framework para layout responsivo, componentes de navegação, modais e utilitários de espaçamento.
- **Google Fonts**: Utilização da fonte 'Poppins' para uma tipografia moderna e legível.
